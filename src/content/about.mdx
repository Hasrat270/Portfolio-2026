## whoami

I'm **Hasrat** — a full-stack developer who builds web applications end-to-end. I care about shipping things that work, code that's easy to change later, and interfaces that respect the user's time.

## what I do

Most days I'm working with **TypeScript, Next.js, Node.js, and PostgreSQL**. I've shipped e-commerce platforms with admin dashboards, multi-tenant SaaS products, and internal tools. I'm equally comfortable in the database, in the API layer, and pushing pixels in the browser.

## how I work

- Start by understanding the actual problem, not the proposed solution
- Prefer boring, proven tools — reach for novelty only when it earns its place
- Ship in small, reversible increments
- Write the code so the next person (often future-me) doesn't curse the past

## outside of code

I read a lot — engineering blogs, systems books, the occasional philosophy detour. I'm always experimenting with a side project; check the [/now](/now) page to see what's current.
